## Run Locally

Clone the project

```bash
  git clone https://github.com/MITHLESH24364/React_E-Commerce.git
```

Go to the project directory

```bash
  cd React_E-Commerce
```

Install dependencies

```bash
  npm install
```
```bash
  npm install react-router-dom
```
```bash
  npm install axios
```

Start the server

```bash
  npm start
```
